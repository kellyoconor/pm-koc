# Quarterly Strategy Review

## Schedule
Last week of every quarter

## What It Does
Full strategic review — evaluates progress against OKRs, reassesses product strategy, reviews market position, and sets direction for the next quarter.

## Workflow
1. **Metrics Review**
   - Pull quarterly metrics vs. targets (north star, input metrics, OKRs)
   - Run cohort analysis on retention and engagement trends
   - Assess financial health (SaaS metrics if applicable)

2. **Strategy Assessment**
   - Review current product strategy for relevance
   - Run PESTLE for macro environment changes
   - Run Porter's Five Forces for competitive dynamics shifts
   - Assess product-market fit trajectory

3. **Discovery Review**
   - Summarize validated/invalidated assumptions from the quarter
   - Review opportunity solution tree — prune and update
   - Surface top unmet customer needs from feedback data

4. **Forward Planning**
   - Draft or update product strategy for next quarter
   - Propose OKRs aligned with updated strategy
   - Identify top bets and their risk profiles
   - Run pre-mortem on proposed Q+1 plan

5. **Multi-Perspective Evaluation**
   - Run ALL subagents on proposed Q+1 plan
   - Synthesize perspectives into go/adjust/pivot recommendation

## Output
Quarterly Strategy Review Package:
- Executive summary (1 page)
- Metrics dashboard with trends
- Strategy assessment (what's working, what's not)
- Updated product strategy (if changed)
- Proposed OKRs for next quarter
- Top 3 bets with risk profiles
- Multi-perspective synthesis
- Decision: stay course / adjust / pivot
